# 作成するに至った動機
    ・昨今の流行病の影響を受けてライブハウスの利用や一時期公園の利用にも制限がかけられたことで表現する、披露する場が狭くなったことを感じて考案しました。
    また、既存のSNSだと情報量が多すぎるため、コミュニティを狭くすることでより多くの人に見てもらえるのではないかと考えました。
